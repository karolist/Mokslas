Добро пожаловать на страницы интерактивного курса обучения

Введение в технологию MMX™

Внимание! В этом учебном курсе используется режим "фреймов" (одновременный показ нескольких страниц в одном окне) и встроенные программы на языке JavaScript. Поэтому Вам нужен браузер с соответствующими функциональными возможностями. Например, вполне подойдет
  • Netscape Navigator* 3.0 или более новая версия;
  • Microsoft Internet Explorer* 3.0 или более новая версия.
Прежде чем начинать работу, убедитесь, что конфигурация браузера позволяет исполнять JavaScript-программы. Для этого воспользуйтесь следующими меню:
  • в Netscape Navigator - меню Options -> Network Preferences -> Languages.
  • в Microsoft Internet Explorer - меню View -> Options -> Security.
  • в русской версии Internet Explorer - меню Вид -> Параметры -> Безопасность.

А теперь начнем изучение технологии MMX!

Welcome to Introduction to MMX™ Technology Tutorial!

To start learning, please use a frame-capable browser that can execute JavaScript programs. For example, you can use
  • Netscape Navigator* 3.0 or newer version, or
  • Microsoft Internet Explorer* 3.0 or newer version.
Before you start, make sure that JavaScript is enabled in your browser.
  • If you use Netscape Navigator, go to Options -> Network Preferences -> Languages.
  • If you use Microsoft Internet Explorer, go to View -> Options -> Security.

Start the tutorial now!
