Для изучения технологии MMX надо иметь представление об архитектуре прошлых поколений процессоров Intel. Желательно также знакомство с языком ассемблера.

Дополнительную информацию Вы можете получить из следующих источников:

  • Intel Architecture MMX Technology. Programmer's Reference Manual (Технология MMX в архитектуре Intel. Руководство программиста).
  • Optimizing Applications for Pentium® Processor Tutorial (обучающая программа "Оптимизация программ для процессора Pentium").
  • Pentium® Processor User's Manual. Volume 3: Architecture and Programming Manual (Процессор Pentium. Руководство пользователя. Том 3: Руководство по архитектуре и программированию).
