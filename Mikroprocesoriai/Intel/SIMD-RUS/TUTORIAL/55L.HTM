В приложениях, связанных с моделированием освещенности объектов, требуется, чтобы коэффициенты ослабления света за счет атмосферных явлений всегда были не меньше единицы.

Для проверки этих условий программе на языке С требуется выполнение операций сравнения и условного перехода для каждого из полученных коэффициентов Att[i]:
if (Att[i] <1)
xmm[i]=1
else
xmm[i]= Att[i]
Если условие выполнено, то в элемент XMM-регистра заносится единица, в противном случае – значение коэффициента ослабления.
