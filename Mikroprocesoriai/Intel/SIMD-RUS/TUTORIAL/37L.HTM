Некэширующие команды предназначены для записи слабо локализованных данных непосредственно в основную память, минуя кэш-память. Такая технология уменьшает "засорение" кэш-памяти ненужными данными, а также вытеснение из нее кэшированных данных с временнóй или пространственной локализацией.

Команда movntps копирует четыре FP-элемента данных из XMM-регистра в память. Команда movntq копирует 64 разряда данных из MMX-регистра в память.

Эти команды не переносят данные в кэш вышестоящего уровня при записи. Для них характерны особенности, аналогичные работе в режиме записи с буферизацией, в частности, слабая упорядоченность обращений в память на запись. Для обеспечения строгой упорядоченности обращений в память на запись используйте команду sfence , которая будет рассмотрена далее в этом уроке.

Выберите movntq или movntps для иллюстрации работы команд.
