SIMD-команды для работы с данными одинарной точности с плавающей запятой (SPFP-данными) используют восемь новых 128-разрядных регистров - XMM-регистров. Команды обращаются к XMM-регистрам по именам: XMM0,...,XMM7.

Эти регистры используются исключительно для работы с данными. Для адресации памяти используются целочисленные регистры и определенные в архитектуре Intel режимы адресации.
