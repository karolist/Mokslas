Перестановка в цикле есть изменение последовательности обращений к данным в теле цикла с тем, чтобы эта последовательность соответствовала порядку, в котором данные размещены в памяти компилятором.

Компиляторы с языка C размещают массивы в памяти строка за строкой. Компиляторы с языка FORTRAN размещают массивы столбец за столбцом.

Обращайтесь к элементам массива в том порядке, в котором они расположены в памяти, с тем чтобы все элементы кэш-линии могли быть использованы доее вытеснения.

При переписывании программ с C на FORTRAN и обратно не забывайте о различии в размещении элементов массивов. В противном случае могут возникнуть существеные потери в производительности кода.
