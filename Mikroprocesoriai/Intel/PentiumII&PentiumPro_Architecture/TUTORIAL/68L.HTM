Чтобы избежать затрат на употребление команды FLDCW, используйте один из следующих приемов:

  • Не используйте FLDCW. Не переключайте режим округления, если этого не требует характер вычислений.
  • Организуйте код так, чтобы операция округления выполнялась в цикле.
  • Применяйте альтернативный алгоритм округления.

    • На данном примере демонстрируется применение первого приема. Округлениев цикле и замена алгоритма округления описаны ниже.

    В приведенных здесь фрагментах кода допустимо использование округления до ближайшего целого. Оптимизированный текст на C содержит вызов функции f2i_CurrentRoundMode() для выполнения округления без переключения режима округления (без выполнения команды FLDCW).

    Выберите ассемблерный код и нажмите на кнопку "Оптимизировать", чтобы посмотреть, как данный прием оптимизации может быть применен к коду на ассемблере.
