Механизм динамического предсказания ветвлений использует историю, хранящуюся в буфере адресов ветвления, для предсказания адреса ветвления. Адрес ветвления есть адрес, на который будет передано управление после выполнения команды ветвления.

Для каждого адреса, хранящегося в буфере адресов ветвления, этот буфер содержит историю четырех последних выполнений команды ветвления, расположенной по этому адресу.

Таким образом, любое ветвление, повторяющее результаты своего выполнения с периодом не более четырех, может быть предсказано.
